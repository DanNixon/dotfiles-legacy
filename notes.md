# Notes

## Setting environment variables in Fish

```
set -Ux [VARIABLE] [VALUE]
```

## Setting Fish PATH

```
set --universal fish_user_paths $fish_user_paths ~/path/name
```

https://github.com/fish-shell/fish-shell/issues/527#issuecomment-32237807
